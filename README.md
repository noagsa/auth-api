# auth-api

REST API with JWT authentication and role-based access control built with Spring Boot.

Built as a portfolio project to demonstrate backend fundamentals: authentication, authorization, and middleware design.

## Tech Stack

- Java 21
- Spring Boot 3.5.11
- Maven
- Docker

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/noagsa/auth-api
cd auth-api
```
2. Copy `application.properties.example` to `application.properties`.
3. Set your JWT secret key (minimum 32 characters)
4. Run the application:

```bash
./mvnw spring-boot:run
